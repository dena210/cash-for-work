
# Cash for Work Platform

## فكرة الموقع
منصة إلكترونية لعرض فرص عمل مؤقتة للشباب، تهدف إلى تخفيف البطالة من خلال توفير فرص عمل قصيرة الأمد بعد الحرب.

## المشكلة
بعاني عدد كبير من الشباب من البطالة وصعوبة الوصول إلى فرص عمل واضحة ومنظمة.

## الحل
إنشاء موقع إلكتروني بسيط يعرض فرص العمل مؤقتة .

## الفئة المستفيدة
الشباب العاطلون عن العمل والباحثون عن فرص عمل مؤقتة.

## أهم الميزات
- عرض فرص العمل باستخدام جدول
- تصفية الفرص حسب المجال
- إظهار حالة الفرصة (متاح / مكتمل)
- نموذج تسجيل مع تحقق من المدخلات

## التقنيات المستخدمة
- HTML5
- CSS3
- Bootstrap
- JavaScript
- jQuery

## طريقة التشغيل
فتح ملف index.html باستخدام المتصفح
